Legacy implementation; see src/clustering.cpp for current version

This is a snapshot of an earlier k-means implementation used for initial feasibility studies. It demonstrates core concepts:

  • Computing encrypted Euclidean distances
  • Initializing centroids from encrypted data
  • Iterative k-means optimization on ciphertexts

Note: The current src/clustering.cpp version is more extensively documented and has been refined based on thesis research and optimization. This legacy version is archived for reproducibility.
