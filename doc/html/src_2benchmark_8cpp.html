Performance Benchmarking for HE-MPC Clustering Pipeline.

MIT License Copyright (c) 2024-2026 Lakshmi R. Kiran Pasumarthy SPDX-License-Identifier: MIT

TIPS-HECluster — Privacy-Preserving Threat Intelligence Clustering Trusted Privacy-Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

This module measures the execution time of key HE operations:

  • Encryption of threat data
  • Homomorphic k-means clustering
  • Decryption of cluster assignments

Uses std::chrono high_resolution_clock for microsecond-precision timing. Output is printed to stdout in human-readable format.

Typical workflow:

  1. Initialize encryption context and key pair
  2. Fetch threat data from API
  3. Time and execute: encryption → clustering → decryption
  4. Report execution costs for each stage
