Interactive Encryption Scheme Selector (CKKS vs BGV).

TIPS-\/\+HECluster — Privacy-\/\+Preserving Threat Intelligence Clustering Trusted Privacy-\/\+Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

This interactive utility demonstrates how to select and initialize different homomorphic encryption schemes based on data type and use case.

Scheme Selection Criteria:\+
\begin{DoxyItemize}
\item CKKS:\+ For floating-\/point threat scores (severity, confidence, normalized time) Supports approximate arithmetic with high precision
\item BGV:\+ For integer-\/based counts (attack frequency, packet counts) Supports exact arithmetic on integers
\end{DoxyItemize}

The program:\+
\begin{DoxyEnumerate}
\item Prompts user to choose data type (floating-\/point or integer)
\item Selects appropriate HE scheme
\item Configures scheme-\/specific parameters
\item Enables required HE capabilities (PKE, KEYSWITCH, LEVELEDSHE)
\item Reports successful setup
\end{DoxyEnumerate}
