SCALE-\/\+MAMBA MPC program for secure argmin computation. This program runs on 3 semi-\/honest parties who cooperatively compute cluster assignments from encrypted distance metrics.

Algorithm:\+
\begin{DoxyEnumerate}
\item Each party inputs a subset of n threat records
\item For each record, compute argmin over k encrypted distances
\item Output final cluster assignment (plaintext)
\end{DoxyEnumerate}

Security:\+ Semi-\/honest adversary, no collusion
