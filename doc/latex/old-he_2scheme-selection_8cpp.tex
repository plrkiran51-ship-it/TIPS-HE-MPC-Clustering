Interactive menu-\/driven utility for selecting between CKKS and BGV schemes. User selects data type, program initializes appropriate context and parameters.

Scheme Rationale:\+
\begin{DoxyItemize}
\item CKKS (Cheon-\/\+Kim-\/\+Kim-\/\+Song):\+ Floating-\/point threat scores, approximate arithmetic
\item BGV (Brakerski-\/\+Gentry-\/\+Vaikuntanathan):\+ Integer counts, exact arithmetic
\end{DoxyItemize}

This archived version demonstrates the early exploration of scheme selection for different threat data types within the research.
