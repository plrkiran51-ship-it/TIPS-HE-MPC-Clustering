Measures end-\/to-\/end execution time for the HE clustering pipeline:\+
\begin{DoxyEnumerate}
\item Initialize cryptographic context and key pair
\item Fetch threat data from Flask API
\item Encrypt threat scores (time this stage)
\item Run homomorphic k-\/means on encrypted data (time this stage)
\item Decrypt cluster assignments (time this stage)
\item Report timing for each stage in microseconds
\end{DoxyEnumerate}

This archived version demonstrates timing instrumentation used for performance analysis and parameter sweep experiments.
