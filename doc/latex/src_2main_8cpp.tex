TIPS-\/\+HECluster — Privacy-\/\+Preserving Threat Intelligence Clustering Trusted Privacy-\/\+Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

This utility program tests the configuration loading mechanism used throughout the HE-\/\+MPC clustering pipeline. It reads the YAML configuration file (../\+config.yaml) and verifies that key HE parameters are correctly parsed.

The program serves as:\+
\begin{DoxyItemize}
\item A bootstrap sanity check during build/\+testing
\item A template for loading config in other modules
\item A diagnostic tool to inspect runtime parameters
\end{DoxyItemize}

Expected config.\+yaml format:\+ openfhe:\+ ring\+\_\+dim:\+ 16384 \# Ring dimension (security parameter) kmeans:\+ n\+\_\+clusters:\+ 3 \# Number of clusters k
