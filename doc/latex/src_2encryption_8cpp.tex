\doxysubsection{Detailed Description}
CKKS Homomorphic Encryption for Threat Scores. 

MIT License Copyright (c) 2024-\/2026 Lakshmi R. Kiran Pasumarthy SPDX-\/\+License-\/\+Identifier:\+ MIT

TIPS-\/\+HECluster — Privacy-\/\+Preserving Threat Intelligence Clustering Trusted Privacy-\/\+Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

Demonstrates complete CKKS encryption workflow for floating-\/point threat scores:\+
\begin{DoxyEnumerate}
\item Fetch threat data from locally-\/running Flask API
\item Initialize Open\+FHE CKKS context with HE parameters
\item Generate public/\+secret key pair
\item Encrypt threat severity scores
\item Perform homomorphic operations (compute mean without decryption)
\item Decrypt results and display
\end{DoxyEnumerate}

Uses libcurl for HTTP communication with Flask backend and nlohmann/\+json for JSON parsing. This module serves as both example code and integration test.

CKKS scheme properties:\+
\begin{DoxyItemize}
\item Supports approximate real/\+complex arithmetic
\item Efficient for floating-\/point threat metrics
\item Allows addition and multiplication of ciphertexts
\item Requires scaling factor management for precision
\end{DoxyItemize}
