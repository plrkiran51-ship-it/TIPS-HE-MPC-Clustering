\doxysubsection{Detailed Description}
Hybrid HE-\/\+MPC K-\/\+Means Implementation (Prototype). 

MIT License Copyright (c) 2024-\/2026 Lakshmi R. Kiran Pasumarthy SPDX-\/\+License-\/\+Identifier:\+ MIT

TIPS-\/\+HECluster — Privacy-\/\+Preserving Threat Intelligence Clustering Trusted Privacy-\/\+Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

Hybrid approach combining:\+
\begin{DoxyItemize}
\item HE (Homomorphic Encryption):\+ Encrypted distance computation via CKKS
\item MPC (Multi-\/\+Party Computation):\+ Encrypted argmin via SCALE-\/\+MAMBA
\end{DoxyItemize}

This prototype demonstrates the feasibility of the hybrid architecture for privacy-\/preserving threat intelligence clustering.

Key Components:\+
\begin{DoxyItemize}
\item CSV readers (threat vectors and ground-\/truth labels)
\item ARI (Adjusted Rand Index) computation for accuracy assessment
\item Centroid extraction for specific clusters
\item Plaintext and encrypted k-\/means workflows
\end{DoxyItemize}

Threat Model
\begin{DoxyItemize}
\item Semi-\/honest adversary (MPC layer):\+ parties follow protocol but may infer data
\item Honest-\/but-\/curious adversary (HE layer):\+ cloud provider cannot decrypt
\end{DoxyItemize}

Scalability:\+
\begin{DoxyItemize}
\item Tested on threat datasets from 10K to 1M records
\item Configurable ring dimensions (8192 to 32768 for N)
\item Batch size tuneable for hardware (1024 to 16384)
\end{DoxyItemize}
