\doxysubsection{Detailed Description}
Performance Benchmarking for HE-\/\+MPC Clustering Pipeline. 

MIT License Copyright (c) 2024-\/2026 Lakshmi R. Kiran Pasumarthy SPDX-\/\+License-\/\+Identifier:\+ MIT

TIPS-\/\+HECluster — Privacy-\/\+Preserving Threat Intelligence Clustering Trusted Privacy-\/\+Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

This module measures the execution time of key HE operations:\+
\begin{DoxyItemize}
\item Encryption of threat data
\item Homomorphic k-\/means clustering
\item Decryption of cluster assignments
\end{DoxyItemize}

Uses std::\+chrono high\+\_\+resolution\+\_\+clock for microsecond-\/precision timing. Output is printed to stdout in human-\/readable format.

Typical workflow:\+
\begin{DoxyEnumerate}
\item Initialize encryption context and key pair
\item Fetch threat data from API
\item Time and execute:\+ encryption → clustering → decryption
\item Report execution costs for each stage
\end{DoxyEnumerate}
