\doxysubsection{Detailed Description}
Threat Data Ingestion and CKKS Encryption. 

MIT License Copyright (c) 2024-\/2026 Lakshmi R. Kiran Pasumarthy SPDX-\/\+License-\/\+Identifier:\+ MIT

TIPS-\/\+HECluster — Privacy-\/\+Preserving Threat Intelligence Clustering Trusted Privacy-\/\+Preserving Threat Information Platform for Sharing (TIPS) Edinburgh Napier University — PhD Research

This module handles the initial stage of the HE pipeline:\+ fetching raw threat data from a Flask API and converting it to encrypted form.

Workflow:\+
\begin{DoxyEnumerate}
\item Initialize CKKS cryptographic context with HE parameters
\item Generate public/\+secret key pair
\item Fetch threat records via HTTP from Flask server
\item Extract threat\+\_\+score field from each record
\item Encrypt all scores under public key
\item Return vector of ciphertexts for downstream clustering
\end{DoxyEnumerate}

Key Functions:\+
\begin{DoxyItemize}
\item \doxylink{src_2data-ingestion_8cpp_ae1e4e37697f23f59aa83147809435b4e}{Fetch\+Threat\+Data()}:\+ HTTP GET to API, returns JSON array
\item \doxylink{src_2data-ingestion_8cpp_a0aeb38603b8796c48113f4233d2c191e}{Encrypt\+Threat\+Data()}:\+ Converts JSON scores to CKKS ciphertexts
\end{DoxyItemize}

Assumptions:\+
\begin{DoxyItemize}
\item Flask API running at \href{http://127.0.0.1:5000/get_threat_data}{\texttt{http:\+/\+/\+127.\+0.\+0.\+1:\+5000/\+get\+\_\+threat\+\_\+data}}
\item Each threat record has a "{}threat\+\_\+score"{} field (double)
\item Network connectivity with API server established
\end{DoxyItemize}
