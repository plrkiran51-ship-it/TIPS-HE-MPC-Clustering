Proof-\/of-\/concept plaintext k-\/means implementation used to establish the baseline for encrypted clustering accuracy comparison.

Functionality:\+
\begin{DoxyEnumerate}
\item Read threat feature vectors from CSV
\item Run standard k-\/means algorithm (plaintext)
\item Read ground-\/truth labels from separate CSV
\item Compute Adjusted Rand Index (ARI) metric
\item Compare with encrypted HE clustering results
\end{DoxyEnumerate}

This unencrypted version is essential for:\+
\begin{DoxyItemize}
\item Validating algorithm correctness
\item Establishing baseline performance metrics ~\newline

\item Computing ARI \texorpdfstring{$>$}{>}= 0.\+85 threshold validation
\item Benchmarking against HE-\/\+MPC overhead
\end{DoxyItemize}
