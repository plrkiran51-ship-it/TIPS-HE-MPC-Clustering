This file demonstrates a complete CKKS encryption workflow:\+
\begin{DoxyItemize}
\item Fetch threat data from Flask API via HTTP
\item Initialize Open\+FHE cryptographic context
\item Generate public/\+secret key pair with evaluation keys
\item Encrypt threat severity scores
\item Compute homomorphic mean (sum then divide)
\item Decrypt and display result
\end{DoxyItemize}

Note:\+ This is archived code from an earlier project phase. The current implementation in \doxylink{src_2encryption_8cpp}{src/\+encryption.\+cpp} is the maintained version.
