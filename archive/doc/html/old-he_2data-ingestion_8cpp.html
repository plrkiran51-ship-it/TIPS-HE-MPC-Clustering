Deprecated
Legacy implementation; see src/data-ingestion.cpp for current version

Implements the data ingestion pipeline:

  1. HTTP GET to Flask API to fetch threat records
  2. CKKS encryption of threat scores
  3. Return encrypted data for clustering

The main() function demonstrates complete workflow: init context, generate keys, fetch data, encrypt, and count successful encryptions.
